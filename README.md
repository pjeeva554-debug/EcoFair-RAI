# EcoFair-RAI

**Responsible AI Evaluation Toolkit (RAI-Toolkit)** is a **Streamlit-based dashboard** that evaluates AI models for:

- **Fairness** – checks if AI predictions are unbiased
- **Transparency** – analyzes feature importance
- **Resource Optimization** – estimates energy consumption and CO₂ emissions
- **Responsible AI Score** – combines model performance, fairness, and energy efficiency

---

## Features

- Works with **categorical and numeric targets** automatically
- Upload any CSV dataset
- Displays:
  - Model Metrics (Accuracy / R² Score)
  - Feature Importance
  - CO₂ Emission & Efficiency
  - Suggestions & Recommendations
  - Responsible AI Score
- Interactive and professional dashboard
